### DESCRIPTION

This project is a modern e-commerce mobile application built using React Native and Expo. It displays a list of products fetched from an external API, supports real-time search, and allows users to view detailed information for each product. The app uses Context API for state management and Expo Router for smooth, file-based navigation, providing a clean and scalable architecture.
